What Is Functional Programming?

Functional programming is considered a programming paradigm, or similarly stated a way of thinking about software construction based on fundamental defining principles. Other programming paradigms you might be more immediately familiar with include object oriented programming or procedural programming.

Functional programing is the process of building software by composing pure functions, having mutable data, shared state, and side-effects. This paradigm is declarative versus imperative, and the state of the application is shared through the pure functions.
